
# Frontend Architecture

DormVibe follows a component-based frontend architecture built with React and TypeScript.

The application is structured around reusable UI components, centralized state management, and page-driven routing to provide a scalable user experience.

---

## Core Pages

### Landing Page

Introduces the platform and guides users into the onboarding process.

### Authentication

Handles user registration and login functionality.

### Personal Quiz

Collects accommodation preferences, lifestyle habits, and social preferences.

### Room Recommendation

Displays personalized room suggestions generated from quiz responses.

### Room DNA

Provides a visual summary of compatibility traits and room preferences.

### Checkout

Simulates accommodation booking and purchase flow.

### Profile Dashboard

Allows users to review recommendations and manage account information.

---

## Design Principles

- Responsive design for desktop and mobile devices
- Reusable React component architecture
- Type-safe development using TypeScript
- State-driven UI updates using Zustand
- Separation of presentation and business logic

---

## Frontend Technologies

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Zustand
- Framer Motion
